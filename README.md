The manuscript titled ``Physics-informed Quadratic Neural Network with Local Feature-aware Positional Encoding for Seismic Waveform Simulation" has been submitted.

We exploit the multiscale localization inherent to wavelet functions and propose a novel Local feature-Aware Positional Encoding (LAPE), which is integrated into PINNs to achieve high-fidelity solutions for scattered wave equations. 

We further derive a physics-driven initialization (PDI) for LAPE's parameters by aligning them with the dominant spatial wavenumbers in the velocity model, ensuring optimal adaptation to local wavefield complexities. 

We introduce quadratic neural networks (QNNs) as replacements for standard MLPs in PINNs, exploiting their theoretically superior approximation properties for nonlinear wave interactions.

The velocity models used in the article are 2000×2000, but since the original file is too large, we have only uploaded a 200×200 model here.
